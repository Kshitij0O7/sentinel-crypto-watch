# Crypto Fun Detector

## Setup

1. Clone Repository.
   ```shell
   git clone https://github.com/anmol-commits/sentinel-crypto-watch
   ```
2. Install depencies.
   ```shell
   npm i
   ```
3. Replace the `<Bitquery Access Token>` placeholder with your access token.
4. Run using the following command and checkout on `localhost:8080`.
   ```shell
   npm run dev
   ```
